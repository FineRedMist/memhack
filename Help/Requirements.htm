System Requirements

  • Windows XP and Windows 2003. It *may* work on others but it is not guaranteed.
  • .NET Framework 1.1 (untested on 1.0)
  • psapi.dll (installed by default for XP and 2003 I believe).

    To edit the memory of a running application you need to have sufficient permissions to do so. You can generally modify any application that you yourself have started. However if you are only a user on the system and not an administrator, you may find that applications that aren't yours you don't have rights to modify in which case the attempt will fail.
